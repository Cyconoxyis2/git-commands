# status
Shows the current state of your files (tracked, untracked, or staged).
# push
Uploads your local commits to a remote repository.
# pull
Fetches changes from a remote server and immediately merges them into your current local branch.
# add
Adds specific file changes to the "staging area" to prepare them for a commit. Use git add . to stage all changes in the current directory.
# commit
Saves a snapshot of your staged changes with a descriptive message.
- -m "Message"
- -a "add all changes"
- -am "combination" 
# branch
Lists all local branches in your repository.
- -a "add all"
- -d "delete"
# merge
Combines changes from another branch into your current active one.
